# Toss Coin Merge (Web)

`docs/primary.md` PRD 기반 · Vite + React + TypeScript + Tailwind CSS

## 스택

- **Core:** React 19, TypeScript, Vite 7
- **Styling:** Tailwind CSS 3 (TDS 스타일: 배경 `#F2F4F6`, Primary `#3182F6`, Pretendard)
- **Physics:** matter-js
- **State:** Zustand
- **Icons:** lucide-react

## 실행

```bash
npm install
npm run dev
```

빌드: `npm run build`  
미리보기: `npm run preview`

## 디자인 (TDS)

- 배경: `#F2F4F6` (`bg-background` / `bg-base`)
- Primary: `#3182F6` (`text-primary` / `bg-primary`)
- 텍스트: `#191F28` (`text-text`)
- 폰트: Pretendard (CDN) + 시스템 폰트 fallback
